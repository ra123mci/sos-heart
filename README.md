## Teste et demmarage

#### Installation des dependances composer : 
```bash
composer install
```

#### Installation des assets npm : 
```bash
npm install
```

#### Demmarage de l'application : 
Mode de develloppement
```bash
npm run dev
```

Mode de production
```bash
npm run build
```
